# @frontendmonster/eslint-config

my personal ESLint rules.

## Installation

```sh
npm install --save-dev eslint @frontendmonster/eslint-config
```

## Usage

Add the extends to your `.eslintrc`:

```javascript
{
  "parserOptions": {
    "project": "<PATH_TO_TSCONFIG>"
  },
  "extends": ["@frontendmonster"],
}
```
